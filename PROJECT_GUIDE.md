## 📦 项目结构

```
binance-trading-scout/
├── scout.js                    # 主程序（真实 API 版本）
├── test.js                     # 演示程序（模拟数据）
├── package.json                # 依赖配置
├── config.example.json         # 配置模板
├── README.md                   # 项目说明
├── SUBMISSION.md               # 完整投稿内容
├── SUBMISSION_SHORT.md         # 简化投稿版本
├── IMAGE_GENERATION_GUIDE.md   # 漫画生成指南

    ├── 05-demo-output.png      # 实际运行截图
    ├── 06-architecture.png     # 架构图
    └── 07-telegram-notification.png  # Telegram 推送示例
```

1. 将内容发布到币安活动页面
