# 🦞 OpenClaw Trading Scout Pro

> AI 驱动的全功能加密货币交易助手

## 🎉 Pro 版本新功能

### 1. 📊 市场情绪分析
AI 分析 24 小时市场数据，判断每个币种的情绪：
- 看涨 🟢 / 震荡 🟡 / 看跌 🔴
- 分析价格、交易量、波动趋势
- 给出操作建议

**示例输出：**
```
📊 市场情绪分析

BTC: 看涨 🟢
- 24h 涨幅: +2.3%
- 交易量增加 15%
- 建议: 持有，目标 $98,000

ETH: 震荡 🟡
- 24h 涨幅: -0.5%
- 交易量持平
- 建议: 观望，等待突破 $2,900
```

### 2. 🔔 智能价格提醒
设置价格提醒，触发时 AI 自动分析并推送：

**使用方法：**
```
你: "提醒我 BTC 跌到 95000"
AI: "✅ 已设置价格提醒"

触发时推送：
🔔 价格提醒
BTC 已跌破 $95,000
当前价格: $94,850

🤖 AI 建议:
- 支撑位: $94,500
- 建议: 可以考虑分批建仓
- 风险: 可能继续下探至 $93,000
```

### 3. 💰 资金流向分析
分析最近 100 笔成交，识别大户动向：
- 净流入 🟢 / 净流出 🔴
- 买卖盘力量对比
- 资金轮动趋势

**示例输出：**
```
💰 资金流向分析

过去 1 小时:
BTC: 净流入 $5.2M 🟢
ETH: 净流出 $2.1M 🔴
BNB: 净流入 $1.8M 🟢

🤖 AI 判断:
- 资金从 ETH 流向 BTC
- 可能是轮动行情
- 建议: 关注 BTC，减持 ETH
```

### 4. 🎯 支撑/阻力位分析
分析订单簿深度，找出关键价位：
- 支撑位（大单买盘）
- 阻力位（大单卖盘）
- 突破/跌破概率

**示例输出：**
```
🎯 BTCUSDT 支撑/阻力位分析

当前价格: $96,500
支撑位: $96,000, $95,500, $95,000
阻力位: $97,000, $97,500, $98,000

🤖 AI 判断:
- 接近阻力位 $97,000
- 突破概率: 60%
- 建议: 等待突破确认再入场
- 止损: $96,200
```

### 5. ⚠️ 异常波动监控
实时监控 5 分钟内波动超过 2% 的币种：
- 自动检测异常波动
- AI 分析原因
- 给出操作建议

**示例输出：**
```
⚠️ 异常波动检测

SOL 5分钟内暴涨 3.2%
当前: $148.50

🤖 AI 判断:
- 可能原因: 大单买入
- 风险: 可能回调
- 建议: 不追高，等待回调至 $145
```

### 6. 📈 历史回测报告
统计历史套利机会，优化策略：
- 总机会数
- 平均价差
- 最佳交易对
- 策略有效性

**示例输出：**
```
📈 策略回测报告

时间范围: 启动至今
监控交易对: BTC, ETH, BNB, SOL

发现机会: 23 个
平均价差: 0.73%
最佳交易对: BNB/ETH (8 次)

💡 建议: 当前策略有效，建议继续使用
```

## 🚀 快速开始

### 安装
```bash
git clone https://github.com/pjl914335852-ux/openclaw-trading-scout.git
cd openclaw-trading-scout
npm install
```

### 配置
```bash
cp config.example.json config.json
nano config.json
```

### 运行
```bash
# Pro 版本（推荐）
node crypto-scout-pro.js

# 或使用 PM2
pm2 start crypto-scout-pro.js --name trading-scout-pro
```
